
  # AI-Powered eBay Store Manager

  This is a modern, AI-powered dashboard for managing your eBay store with integrated Material Dashboard from Creative Tim. The application connects to your actual eBay store data through the eBay Developer APIs and provides real-time insights, analytics, and management tools.

  ## ✨ Features

  - **Material Dashboard Integration**: Beautiful, responsive Material Dashboard UI from Creative Tim
  - **Live eBay Data**: Real-time connection to your eBay store via APIs
  - **AI-Powered Insights**: Intelligent recommendations and analysis
  - **Order Management**: Track and manage orders in real-time
  - **Listing Analytics**: Monitor active listings and performance metrics
  - **Modern React Architecture**: Built with React, TypeScript, and modern UI components

  ## 🛠️ Material Dashboard Integration

  The application now includes the complete Material Dashboard from Creative Tim, featuring:

  - **Professional Sidebar Navigation**: Clean, intuitive navigation with Material icons
  - **Live Data Cards**: Real-time metrics display (Revenue, Orders, Listings, Conversion Rate)
  - **Interactive Charts**: Chart.js integration for data visualization
  - **Responsive Tables**: Orders and store data display
  - **Modern UI Components**: Material Design components throughout

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  The dashboard will be available at `http://localhost:3000` with the Material Dashboard as the default view.
